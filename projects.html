Leaflet & Javascript

click header

Using the URL from USGS for the JSON earthquake data, I pulled the data in to visualize. After data was sourced I then proceeded to use leaflet to plot all of the earthquakes based on the longitude and latitude. Using Javascript the data markers on the map reflect the magnitude of the earthquake by their size and the depth of the earthquake by color. Earthquakes with higher magnitudes appear larger and those with greater depth appear darker in color. Also, added popups to display more information on the selected earthquake on the map (magnitude and depth). To add a bit more of a user customized experience I added a number of base maps to choose from as well as separated the 2 data sets (earthquakes and tectonic plates) into overlays that can be turned on and off independently.
